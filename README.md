# 📝 App ToDo List
Projeto mobile, cujo objetivo era criar uma lista de tarefas completa.


## 💻 Tecnologias
- React Native
- TypeScript
  

## ⚙️ Funcionalidades
- Cadastro de tarefas.
- Remoção de tarefas.
- Marcar uma tarefa como concluída.
- Verificar a quantidade de tarefas criadas.
- Verificar a quantidade de tarefas concluídas.
